Installeer de app op je beginscherm zodat je API key permanent wordt opgeslagen

1
Tik het Deel-icoon □↑ onderaan in Safari
2
Scroll naar beneden en tik "Zet op beginscherm"
(niet "Voeg toe aan favorieten")
3
Tik rechtsboven op "Voeg toe"
4
Open de app altijd via het beginscherm-icoon — nooit via Safari direct
⚠️ Belangrijk: Open je de app via Safari of een bookmark? Dan wordt de API key gewist zodra je de browser sluit. Via het beginscherm-icoon blijft alles bewaard.
